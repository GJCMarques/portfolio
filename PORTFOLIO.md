### 1. ALOJAMENTO GI — *"A Casa do Gi"*
> **Plataforma de Reservas e E-commerce para Alojamento Local**

**Tipo:** Comercial (cliente pago) · Produção ativa

**Descrição:**
Website completo para um alojamento local em Mogadouro, com sistema de reservas, carrinho de compras, gestão de faturas e pagamentos online integrados. Projeto entregue a cliente real, encontrando-se atualmente em produção.

**Funcionalidades principais:**
- Sistema de reservas com gestão de disponibilidade
- Carrinho de compras com produtos e serviços do alojamento
- Integração com gateway de pagamentos **IfthenPay** (referências MB, Cartão de Crédito)
- Geração automática de faturas com código de barras, UUID único e hash **HMAC-SHA256** para verificação de autenticidade
- Envio automático de emails transacionais com **PHPMailer**
- Painel de administração completo para gestão de reservas, produtos e clientes
- Proteção CSRF e autenticação segura
- Design responsivo e adaptado a dispositivos móveis
- Suporte multilingue (Português / Inglês)

**Tecnologias utilizadas:**

| Camada | Tecnologias |
|--------|-------------|
| Backend | PHP 8.0+, Composer |
| Base de Dados | MySQL |
| Frontend | HTML5, CSS3, Tailwind CSS, JavaScript |
| Email | PHPMailer |
| Pagamentos | IfthenPay API |
| Segurança | CSRF, HMAC-SHA256, UUID |

  12. CRISTAL CAPITAL TERMINAL

  Terminal Financeiro Profissional — Plataforma Quant de Alto Desempenho

  Tipo: Pessoal / Showcase Técnico · Em desenvolvimento ativo

  Descrição:
  Terminal financeiro de nível profissional inspirado no Bloomberg Terminal, desenvolvido como plataforma completa de análise de mercados, quantitative
   finance e IA. Arquitectura SPA de rota única com design IBM Plex Mono em estética escura âmbar, construída sobre Next.js 15 com React 19 e estado   
  global gerido por Zustand 5. O projeto demonstra domínio de tecnologias de ponta em finanças quantitativas, integração de IA local e sistemas de     
  autenticação empresarial.

  Funcionalidades principais:
  - Interface de terminal Bloomberg com 8 temas visuais e paleta de comandos (Ctrl+K)
  - Sistema de autenticação completo com next-auth v5, roles (ADMIN, ANALYST, TRADER, VIEWER) e MFA via TOTP (compatível com Google Authenticator)     
  - Ambiente Quant tri-camada: cálculo TypeScript no browser + Python server-side (numpy, scipy, pandas) + módulos compilados em C++ → WebAssembly     
  (Emscripten)
  - Modelos financeiros avançados: Black-Scholes com gregas completas, Monte Carlo GBM, Markowitz (fronteira eficiente), GARCH(1,1), Heston, SABR,     
  Black-Litterman, Nelson-Siegel
  - Precificação de obrigações (bond pricer): YTM, Z-Spread, duration, DV01, análise de cenários
  - Chat institucional em tempo real com SSE (Server-Sent Events), canais temáticos e presença online
  - Integração com Ollama/Llama 3 para IA financeira local (sem dependências de nuvem)
  - Gráficos interativos com Recharts e lightweight-charts
  - Suporte multilingue completo (PT / EN / ES) com formatadores Intl
  - Base de dados SQLite local — 100% autónomo, sem serviços externos obrigatórios
  - Layout redimensionável com painéis independentes por vista
