## 🎨 Sobre o Projeto

Este website não é apenas uma demonstração dos meus trabalhos; é, por si só, uma prova do meu nível técnico. 

A arquitetura, o design da interface (UI), as experiências de utilizador (UX) e todo o código fonte foram **100% idealizados e desenvolvidos por mim**, sem recurso a construtores visuais de websites ou templates pré-feitos. A minha filosofia foca-se na harmonia entre a Estética (Aesthetics) e a Performance (Engeneering).

## ✨ Destaques e Funcionalidades

- **Desenvolvimento "From Scratch"**: Programado de raiz com foco na pureza do código para garantir máxima performance.
- **Design de Classe Mundial (Award-Winning UX)**: Abordagem visual focada em reter a atenção do utilizador com micro-interações, tipografia moderna e layouts expansivos.
- **Transições SPA (Single Page Application)**: Navegação fluida entre páginas sem carregamentos visíveis, com animações cinematográficas de saída e entrada.
- **Performance Otimizada**: Conversão dinâmica de todos os assets visuais para `.webp`, otimização no carregamento (`Intersection Observers`) e redução do impacto no DOM.
- **Design Responsivo Extremo**: Preparado não só para Telemóveis e Tablets, mas até para rácios de ecrã altamente incomuns em computadores, não quebrando o design em nenhuma situação.

## 🛠️ Tech Stack

Para assegurar o controlo total sobre cada pixel e cada animação, o portfólio baseia-se num stack minimalista e nativo:

- **Frontend Core**: HTML5 Semântico, Vanilla CSS (CSS3) e Vanilla JavaScript (ES6+).
- **Animações**: Custom Keyframes, Intersections Observers dinâmicos e GSAP-like custom scripting.
- **Design Visual**: Desenho estrutural e conceptual (Figma), manipulação fotográfica e vetorial.
